# AI Mily Medical Interpreter — Case Study

## Project Overview
AI Mily Medical Interpreter was a real-time healthcare interpretation platform designed to support clinicians and patients through live voice translation and dual-language transcription. The project focused on rebuilding and improving an existing medical translation web application with a modern frontend, a Python-based backend, and a stronger emphasis on usability, security, and privacy.

This project was completed as an industry-facing academic team project at RMIT University.

---

## My Role
**Frontend Developer | Backend Contributor | Quality Assurance | Scrum Master (Sprint 4)**

I contributed across frontend development, backend-related implementation, and QA. My work included helping deliver a responsive and accessible user interface based on Figma designs, supporting backend-related functionality in Python and AI integration workflows, and contributing to testing, issue tracking, and Agile sprint execution.
